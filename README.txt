# 桌面操作自动重放工具

## 项目介绍

桌面自动重放工具是一个功能简单的桌面操作录制和回放工具，允许用户录制鼠标和键盘操作，然后自动重复执行这些操作，提高工作效率。

## 主要功能

- **操作录制**：录制鼠标移动、点击和键盘按键操作
- **操作回放**：自动回放录制的操作序列
- **循环播放**：支持设置循环播放次数
- **播放速度调整**：可根据需要调整播放速度
- **序列管理**：保存、加载和删除操作序列
- **热键控制**：支持使用Esc键组合键停止录制或播放
- **安全机制**：鼠标移到屏幕角落可停止播放，确保操作安全

## 系统要求

- Python 3.7+
- 所需Python库：
  - PyQt5
  - pynput
  - pyautogui

## 安装方法

1. **克隆或下载项目**

2. **安装依赖**
   ```
   pip install -r requirements.txt
   ```

3. **运行应用程序**
   ```
   python main.py
   ```

## 使用方法

### 录制操作

1. 点击"开始录制"按钮
2. 执行需要自动化的操作
3. 点击"停止录制"按钮或按Esc键停止录制
4. 录制完成后，可选择保存操作序列

### 播放操作

1. 选择要播放的操作序列（或使用当前录制的序列）
2. 设置播放速度和循环次数
3. 点击"播放"按钮开始自动执行操作
4. 如需停止，点击"停止"按钮或移动鼠标到屏幕角落

### 管理序列

- **保存序列**：输入序列名称，点击"保存"按钮
- **加载序列**：从下拉列表选择序列，点击"加载"按钮
- **删除序列**：从下拉列表选择序列，点击"删除"按钮

## 注意事项

1. 录制操作时，请确保操作环境稳定，避免干扰
2. 播放操作时，请勿手动操作鼠标和键盘，以免影响自动化执行
3. 对于需要精确定位的操作，建议在录制时保持屏幕分辨率和窗口位置不变
4. 复杂操作可能需要多次调整和测试，以达到最佳效果

## 日志系统

应用程序会在`logs`目录生成日志文件，记录操作执行情况和错误信息，便于排查问题。

## 快捷键

- **开始录制**：点击"开始录制"按钮
- **停止录制**：点击"停止录制"按钮或按Esc键
- **开始播放**：点击"播放"按钮
- **停止播放**：点击"停止"按钮或移动鼠标到屏幕角落

## 故障排除

1. **无法启动应用程序**：检查Python环境和依赖库是否正确安装
2. **录制失败**：确保没有其他程序占用鼠标或键盘
3. **播放不准确**：检查录制时的屏幕分辨率和窗口位置是否与播放时一致
4. **操作执行失败**：查看日志文件获取详细错误信息
